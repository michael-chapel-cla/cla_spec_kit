# Roadmap — LiquidityEventOS

## Phase 1
- grant inventory
- scenario comparison
- risk checklist
- planning memo export

## Phase 2
- document ingestion
- collaboration workspace
- trust / charitable strategy modules

## Phase 3
- post-liquidity balance-sheet planning  
- banker / legal / tax coordination workflows
- founder family office lite features

## Phase 6 (Future)
- multi-currency support (EUR, GBP, CAD for cross-border exercises)  
- RSU / ESPP / option grant type expansion (currently ISO/NSO only)
- exercise window tracking with deadline alerts  
- 83(b) election tracker and filing reminders  
- AMT credit recovery tracking (Form 8801)
- secondary sale / tender offer planning  
- concentration risk dashboard with diversification targets  
- tax lot tracking for partial exercises / sales
- vesting schedule visualization (4-year cliff/1-month)  
- document OCR pipeline for 409A valuations, cap tables
- LLM-powered risk detection (missing risks, next-best-action)  
- advisor portal with client portfolio overview
- export to CSV / PDF for CPA review

## Phase 7 (Future)  
- mobile app (React Native) for founder-facing interface
- webhooks for deadline / vesting event notifications  
- SSO / SCIM provisioning for firm-level deployments
- SOC 2 compliance (audit logs, data retention policies)  
- multi-language support (English, Spanish, Mandarin)
