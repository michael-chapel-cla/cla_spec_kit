# MVP Build Spec — LiquidityEventOS

## Goal
Help founders and advisors compare key liquidity-event scenarios and identify planning moves before deadlines.

## Stack suggestion
- web app
- structured equity data model
- deterministic equity-tax scenario engine
- LLM summarization layer

## MVP workflows
1. Enter grants and company details
2. Select possible event paths
3. Compare tax / cash outcomes
4. Surface risks and missed opportunities
5. Generate planning memo

## Guardrails
- versioned rule engine
- assumptions disclosed clearly
- advisor/client review before action
