# ExpenseFlow

**Repo name:** `expense-flow`

A SaaS platform for employee expense submission, manager approval, and finance reporting. Employees photograph receipts and submit expenses on mobile or web; managers approve or reject with a single action; finance teams export clean data to payroll and accounting systems.
