# Spec Kit — Hermes Agent Skills

This directory contains four reusable skills for running the spec-kit workflow in Hermes Agent. Each skill encapsulates one command from the three-stage scaffold pipeline:

| Skill | Command | Purpose |
|---|---|---|
| `spec-kit-design` | `/design <app-name>` | Generate 16 requirement documents from `ideas/<app-name>/ideas.md` |
| `spec-kit-plan` | `/plan <app-name>` | Generate an implementation plan from the requirements docs |
| `spec-kit-create` | `/create <app-name>` | Scaffold a full app (backend, db, frontend, helm) from the plan |
| `spec-kit-validate` | `/validate <app-name>` | Audit a generated app against coding standards and produce a scored report |

## Prerequisites (in order)

1. **Ideas** must exist at `/ideas/<app-name>/ideas.md` (before `/design`)
2. **Requirements** must be generated by `/design` (before `/plan`)
3. **Plan** must exist at `/plans/<app-name>/PLAN.md` (before `/create`)
4. **App** must be scaffolded by `/create` (before `/validate`)

## Tech Stack (Fixed — Do Not Deviate)

- **Backend**: Node.js 20 + Fastify + TypeScript
- **Frontend**: React 18 + Vite + TypeScript + MUI v6
- **Database**: MSSQL + Flyway migrations
- **Auth**: Azure Entra (OIDC for SPA, JWT Bearer for API)
- **Infra**: AKS + Helm

## How to Install

### Option A — Manual copy (recommended for local-only use)

```bash
# From repo root:
cp -r hermes-skills/spec-kit-* ~/.hermes/profiles/cheap_chappie/skills/

# Then reload skills so Hermes picks them up:
/hermes reload-skills
```

### Option B — Install from local paths (if Hermes supports it)

```bash
hermes skills install ./hermes-skills/spec-kit-design/SKILL.md --name spec-kit-design
hermes skills install ./hermes-skills/spec-kit-plan/SKILL.md --name spec-kit-plan
hermes skills install ./hermes-skills/spec-kit-create/SKILL.md --name spec-kit-create
hermes skills install ./hermes-skills/spec-kit-validate/SKILL.md --name spec-kit-validate

/hermes reload-skills
```

### Option C — Clone into profile (for shared use)

```bash
cd ~/.hermes/profiles/cheap_chappie/skills/
cp -r ../../hermes-skills/spec-kit-* .
/hermes reload-skills
```

## How to Use

Once installed, load a skill in-session:

```bash
# Load the design skill
/skill spec-kit-design

# Then provide context:
design MyApp
```

Or invoke directly if your Hermes version supports slash commands bound to skills. The skill's embedded instructions will guide you through each step of the workflow.

## Important Constraints

- Do NOT modify `/templates/`, `/specs/`, or `/helm/`
- Always start from templates — never create backend/frontend files from scratch
- Content must be specific to the app — no placeholder text
